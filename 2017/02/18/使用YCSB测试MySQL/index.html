YCSB（Yahoo! Cloud Serving Benchmark）是雅虎开源的一款基准测试工具，目标是通过开发框架和常用数据集，以用来测试不同的Key-Value存储和云服务的性能

Github：https://github.com/brianfrankcooper/YCSB
Wiki：https://github.com/brianfrankcooper/YCSB/wiki
Latest version：https://github.com/brianfrankcooper/YCSB/releases/
